### VM 部署（Variant: vm）

```
172.24.40.31 (client)
     │  go-tpc → :4000
     ▼
172.24.40.34  HAProxy :4000
     │  roundrobin
     ├─▶ 172.24.40.32  TiDB + PD + TiKV
     └─▶ 172.24.40.33  TiDB + PD + TiKV
         172.24.40.34            PD + TiKV

processes: 直接跑於 OS，無容器層
TiKV RF=3，每節點 30 GB data dir
```

### K8s 部署（Variant: k8s-unlimit / k8s-limit）

```
172.24.40.31 (client)
     │  go-tpc → :30004
     ▼
172.24.40.32  NodePort :30004
     │
     │  k3s cluster
     ├─ poc-1  k3s server  ┐
     ├─ poc-2  k3s agent   ├─ TiDB Operator v1.6.5
     └─ poc-3  k3s agent   ┘
          │
          ├─ PD   × 3  req: 500m/1Gi   limit: 1c/2Gi   PV: 10Gi  (local-path)
          ├─ TiDB × 2  req: 500m/1Gi   limit: 1c/3Gi
          └─ TiKV × 3  req:   1c/4Gi   limit: 2c/8Gi   PV: 100Gi (local-path)

k8s-unlimit: 不設 limits（移除 limit 欄位）
k8s-limit  : limits 如上表（反映實際資源上限）
overhead   : k3s control plane 跑於 poc-1，佔約 1 vCPU / 1~2 GB RAM（納入比較）
```
