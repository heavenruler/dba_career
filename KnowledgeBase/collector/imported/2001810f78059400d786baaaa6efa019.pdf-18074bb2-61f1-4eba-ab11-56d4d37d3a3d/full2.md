# sqlmap

sqlmap is an open source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws and taking over database servers. It comes with a powerful detection engine, many niche features for the ultimate penetration tester, and a broad range of switches including database fingerprinting, data fetching from the database, accessing the underlying file system, and executing commands on the operating system via out-of-band connections.

## Installation

You can download the latest tarball or zipball from the releases page:
https://github.com/sqlmapproject/sqlmap/releases

Preferably, download sqlmap by cloning the Git repository:

```
git clone --depth 1 https://github.com/sqlmapproject/sqlmap.git sqlmap-dev
```

sqlmap works out of the box with Python versions 2.6, 2.7 and 3.x on any platform.

## Usage

To get a list of basic options and switches use:

```
python sqlmap.py -h
```

To get a list of all options and switches use:

```
python sqlmap.py -hh
```

You can find sample runs and demonstrations in the project's wiki and YouTube channel. For an overview of sqlmap capabilities, a list of supported features, descriptions of all options and switches, and examples, consult the user's manual.
