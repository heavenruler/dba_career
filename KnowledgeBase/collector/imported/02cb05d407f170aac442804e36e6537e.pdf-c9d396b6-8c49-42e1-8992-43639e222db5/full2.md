作者：阿然（Linux运维进阶之路）  
来源：原创，2025-09-16（上海）

大家好，今天分享一个宝藏项目——PandaWiki。作为运维，我深知文档管理的痛苦：资料散落、搜索效率低、团队协作困难。PandaWiki 能把 AI 能力深度集成到知识管理中，提供智能创作、智能问答和语义搜索，能显著提升文档管理效率。

## 什么是 PandaWiki？

PandaWiki 是一款由长亭科技开源的 AI 大模型驱动的知识库搭建系统，在 GitHub 上有较高关注度。主要特点包括：

- AI 辅助创作：智能生成文档内容和建议  
- AI 智能问答：对话式查询知识库  
- AI 语义搜索：理解搜索意图，精准定位信息

它不仅是文档管理工具，也像一个会思考的智能助手。

## 环境准备

在开始安装之前，确保服务器满足以下要求：

硬件配置：
- 内存：至少 2GB（推荐 4GB 以上）
- 磁盘：至少 1GB 可用空间
- CPU：1 核心以上

系统要求：
- 操作系统：Linux（推荐 Ubuntu 或 CentOS）
- Docker 版本：20.x 以上
- 网络：能正常访问外网

权限要求：
- 需要 root 权限进行安装

使用 root 权限登录服务器，然后执行一键安装命令（脚本会检查环境、下载 Docker 镜像、创建数据目录并启动服务，整个过程约 3–5 分钟）：


## 创建你的第一个知识库

### 新建知识库
1. 在控制台首页点击“创建知识库”。  
2. 填写知识库信息：
   - 名称：例如 “Linux 技术文档库”  
   - 描述：简单说明用途  
   - 访问权限：公开或私有  
3. 点击“创建”完成。

### 添加文档内容
PandaWiki 提供多种方式填充知识库：

方式一：手动创建
- 点击“新建文档”
- 使用富文本或 Markdown 编辑器编写内容
- AI 会根据上下文智能建议和补全

方式二：URL 导入
- 选择“从 URL 导入”
- 输入网页地址，系统会自动提取并格式化内容

方式三：文件导入
- 支持上传 Word、PDF、Markdown 等格式
- 支持批量导入，自动解析文档结构和格式

## 发布与使用

### 发布知识库
- 在知识库设置中找到“发布配置”
- 获取知识库访问链接
- 可自定义域名和主题样式

### 核心功能体验
- AI 智能问答：在知识库前台页面的搜索框中输入自然语言问题，AI 会基于知识库内容给出回答。  
- 第三方集成：支持企业 IM（钉钉、飞书、企业微信）集成，配置 webhook 后可在群聊中 @ 机器人查询知识。
